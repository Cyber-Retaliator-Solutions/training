Lab Requirements for IBM Courses

General Lab Requirements

  • Hardware:
    • IBM Z Mainframe (latest model preferred) with sufficient capacity to support multiple simultaneous users.
    • Adequate memory and storage to handle the course load and data sets.
    • Network connectivity for remote access if required.
  • Software:
    • z/OS Operating System (latest version preferred).
    • IBM z/OS Management Facility (z/OSMF).
    • IBM TSO/E and ISPF for user interfaces.
    • JES2 or JES3 for job entry.
    • Required software licenses for all IBM products used in the courses.
  • Network:
    • High-speed, reliable internet connection.
    • Secure VPN access for remote participants.
    • Firewall configurations to allow necessary protocols and services.

IBM z/OS Bootcamp

User Accounts:

  • Separate user accounts for each participant.
  • Adequate permissions to access and execute training materials.

Storage:

  • Pre-configured data sets and libraries.
  • Adequate DASD space for exercises.

Fundamental System Skills in z/OS

Software:

  • z/OS system configured with basic system libraries and data sets.
  • JCL libraries and sample jobs pre-loaded.

Environment:

  • TSO/E and ISPF configured for each participant.
  • Access to system logs and job outputs for analysis.

IBM System z Hardware Management Console (HMC) Operations

Hardware:

  • Access to an IBM System z server with HMC.
  • HMC console setup and configured for training.

Accounts:

  • User accounts with appropriate HMC access levels.

Hardware Configuration and Definition (HCD) for z/OS

Software:

  • HCD software installed and configured.
  • Access to IODF (Input/Output Definition File) data sets.

Environment:

  • Participants need authority to create and modify HCD configurations.

Introduction to Storage

Hardware:

  • Access to storage devices configured with IBM Z.

Software:

  • Storage management software installed and accessible.

Data:

  • Pre-defined storage pools and volumes for exercises.

Additional Notes

  • Ensure that all participants have secure and reliable access to the lab environment throughout the course duration.
  • Provide a technical contact or support team to address any issues during the training sessions.
  • Schedule a pre-training session to verify that all configurations and setups meet the course requirements.
